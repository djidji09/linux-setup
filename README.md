# i3wm
### install i3 window manager
``` terminal
sudo apt install i3
```
### install the top bar (polybar)
``` terminal
sudo apt install polybar
```
### install a terminal file manger (ranger)
``` terminal
sudo apt install ranger
```
### install an application launcher (rofi)
``` terminal
sudo apt install rofi
```
